# Helix - Triple Helix Innovation

A Python project to promote knowledge economy innovation through the Triple Helix model.

## About

This project focuses on promoting knowledge economy innovation using the Triple Helix framework, which involves collaboration between:
1. Academia (Universities and Research Institutions)
2. Industry (Private Sector)
3. Government (Public Sector)

## Installation

### Desktop/Server Installation

Standard Python environment:
```bash
git clone https://github.com/yabo1234/helix.git
cd helix
python triple-helix-innovation.py
```
