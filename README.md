# Helix - Knowledge Economy Innovation Framework

## Overview

This repository explores innovation in knowledge-based economies through the lens of the **Triple Helix Theory**, which examines the dynamic interactions between University, Industry, and Government.

## Triple Helix Theory

The Triple Helix model describes how innovation emerges from the interplay of three institutional spheres:

1. **University** - Knowledge production and human capital development
2. **Industry** - Product and service innovation, commercialization
3. **Government** - Regulation, funding, and strategic direction

### Theoretical Perspectives

This repository provides comprehensive documentation on Triple Helix theory from two complementary perspectives:

#### Neo-Institutional Perspective
Emphasizes institutional arrangements, organizational structures, and the evolution of rules and norms that enable innovation collaboration.

#### Neo-Evolutionary Perspective
Focuses on dynamic co-evolution, variation-selection-retention mechanisms, and self-organizing innovation systems.
