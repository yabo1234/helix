Qajeelfama (How to use)

Menu irratti qoodinsa (dropdown) jiran cuqaasi/tap godhi. Fuulli kun mobile irratti salphatti akka hojjetuuf JS fayyadama.

1) Menu bana

“Galmee / Haala / Oduu / Gargaarsa” irratti cuqaasi.

2) Link filadhu

Tarree keessaa link filadhu; link alaa (external) fuula haaraa keessatti banama.

3) Cufi

Bakki duwwaa irratti cuqaasi yookaan Escape dhiibi.

Yoo GitHub Pages keessatti jijjiiramni hin mul’anne, hard refresh godhi (Ctrl+F5 / Cmd+Shift+R).
