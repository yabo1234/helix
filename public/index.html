🔬 Triple-Helix Innovation Chatbot

A lightweight AI assistant that helps you structure innovation challenges using the Academia × Industry × Government framework.

🎓 Academia 🏭 Industry 🏛️ Government

What the chatbot does

The chatbot classifies your innovation question into one of five intent categories — funding, commercialisation, policy, research, or partnership — then returns a structured Triple-Helix response with tailored clarifying questions to keep the conversation productive.

🎓 Academia

  • Knowledge & evidence
  • Prototypes & publications
  • Research de-risking

🏭 Industry

  • Market adoption path
  • ROI & incentives
  • IP & commercialisation

🏛️ Government

  • Policy & regulation
  • Public procurement
  • Funding & grants

Example prompts to try

  • "Help me design a university–startup–government pilot for renewable energy forecasting."
  • "How do I structure a consortium for an AI healthcare product while staying compliant?"
  • "I want funding for a smart agriculture prototype. What should the proposal include?"

How to run the chatbot locally

  1. Clone the repository 
    git clone https://github.com/yabo1234/helix.git
cd helix
  2. Create a virtual environment & install dependencies 
    python -m venv .venv
source .venv/bin/activate   # Windows: .venv\Scripts\activate
pip install -r requirements.txt
  3. Start the chatbot 
    python app.py
  4. Open in your browser
    Gradio prints a local URL, e.g. http://localhost:7860. Open it and start chatting!

Tech stack

  • Gradio ≥ 5 — chatbot UI (Python)
  • triple_helix_engine.py — rule-based intent triage (no external APIs required)
  • Netlify — static hosting for this showcase page

The Python chatbot itself runs on any machine with Python 3.9+. To deploy it publicly (not just this page), consider Hugging Face Spaces or Render, both of which support Python/Gradio apps for free.
